


# 🚀 CRUD App - Full-Stack Development  

This is a **full-stack CRUD (Create, Read, Update, Delete) application** built using the **MERN (MongoDB, Express.js, React.js, Node.js) stack**. The project demonstrates how to integrate frontend and backend seamlessly, manage data, and implement **RESTful APIs**.

## 🌍 Live Demo  
🔗 **[View the App](https://quick-manage-garden.lovable.app/)**  

## 🛠️ Tech Stack  
- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **API:** RESTful API using Express.js  
- **Styling:** CSS  

## 🔹 Features  
- Create new entries 📝  
- Read and display data 📄  
- Update existing records ✏️  
- Delete data permanently ❌  
- Fully responsive UI for smooth user experience  

## 🚀 Getting Started  

### 1️⃣ Clone the Repository  
```bash
git clone https://github.com/your-username/crud-app.git
cd crud-app
```

### 2️⃣ Install Dependencies  
```bash
npm install
```

### 3️⃣ Start the Backend Server  
```bash
cd backend
node server.js
```

### 4️⃣ Start the Frontend  
```bash
cd frontend
npm start
```

## 📌 API Endpoints  

| Method | Endpoint         | Description                 |
|--------|-----------------|-----------------------------|
| GET    | `/api/items`    | Get all items              |
| POST   | `/api/items`    | Add a new item             |
| PUT    | `/api/items/:id`| Update an existing item    |
| DELETE | `/api/items/:id`| Delete an item             |
